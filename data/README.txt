This directory should contain the stock data CSV files required for ML4T exercises.

Required files:
- AAPL.csv (Apple stock data)
- IBM.csv (IBM stock data)

Please download these files from:
- Canvas (for enrolled students): ML4T Development Environment setup instructions
- Historical course webpage (for non-enrolled visitors): Current Semester -> Software Setup and Local Environment

The CSV files should have columns: Date, Open, High, Low, Close, Volume, Adj Close
